### projects

**SpectreC2** — Encrypted C2 framework. AES-256-GCM beaconing over HTTP/JSON, REST API for real-time task management, multi-platform implant architecture with modular command routing.

**Shroud** — Windows PE crypter. Section-level XOR encryption with a polymorphic loader stub. The stub is appended to the target binary and handles runtime decryption before transferring execution to the original entry point.

**Pentest Toolkit** — Six offensive security modules:
- Async TCP port scanner with 104 service signature fingerprints
- PE structure analyzer (headers, sections, import/export tables)
- Raw socket packet sniffer with Ethernet/IP/TCP/UDP/ICMP parsing
- Shellcode injector supporting local thread, callback, and remote process techniques
- Exploit development utilities: cyclic pattern generator, ROP gadget search, bad character analysis, XOR/ADD shellcode encoder, buffer builder
- Cryptography toolkit: hash identifier (15 types), wordlist cracker, XOR bruteforce with crib dragging, base64/32/16 detection and decoding
